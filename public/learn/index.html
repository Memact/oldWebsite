Memact helps apps personalize better with context users control.

Basics

What is Memact?
Memact lets apps remember useful context with your permission, instead of each app guessing silently.
Why would an app use Memact?
Apps use Memact to send useful signals, save context, and run features that make the app feel more personal.
What can users control?
Users choose which apps connect, what those apps can use, and whether access continues.

Developers

How does an app connect?
Register an app, keep the API key on your server, add Connect Memact, then use the SDK/API to send capture events and run features after access is approved.
What does the SDK do?
The SDK helps apps verify access, send capture events, list features, run features, and retrieve permitted schema or memory summaries.
What is Studio?
Studio is the open-source feature space where contributors can add Memact features inside the SDK and Contracts boundaries.
